user-info-management

Project Description

This is a simple Java console application that prompts the user for some basic information (name, age, phone number, email) and saves it to a CSV file. The program also includes a function to read the name of a specific row from the CSV file.

How to Run

  1. Download and install Java.
  2. Download the project files from GitHub.
  3. Open a terminal and navigate to the project directory.
  4. Compile the Java files: javac *.java
  5. Run the program: java App

Project Structure

  • App.java: the main Java file that contains the user input and file I/O functions.
  • data.csv: the CSV file where the user data is saved.

Dependencies

  • OpenCSV: a third-party library for reading and writing CSV files in Java.
